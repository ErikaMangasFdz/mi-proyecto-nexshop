# Proyecto NexShop

Base de datos relacional para la gestión de ventas, sedes, productos y empleados de la empresa NexShop.

## Descripción del Proyecto
Este proyecto consiste en el diseño, implementación y carga de datos de una base de datos MySQL. Se ha realizado siguiendo el ciclo de vida de desarrollo de software, desde el diseño del modelo Entidad-Relación hasta la ejecución de consultas SQL complejas.

## Contenido del Repositorio
- **/sql/**: Contiene los scripts necesarios:
  - `schema.sql`: Estructura y creación de las tablas.
  - `datos.sql`: Inserción de los datos de prueba.
- **/consultas/**: Archivo `consultas.sql` con las sentencias para la extracción de información.
- **/docs/**: Documentación visual que incluye:
  - `diagrama_er.png`: Diagrama Entidad-Relación de la base de datos.
  - `resultado_consulta.png`: Captura de pantalla de la ejecución del JOIN principal.

## Tecnologías Utilizadas
- **Base de Datos:** MySQL
- **Herramienta de gestión:** DBeaver
- **Control de versiones:** Git / GitHub
